# Pomodoro Timer

A sleek, glassmorphic productivity timer built with HTML, CSS, and JavaScript. It helps you stay focused using the Pomodoro technique: 25 minutes of work followed by a 5-minute break, with a long break every four sessions.

## Features
- **Work/Break Cycles:** Automatic switching between modes.
- **Session Tracking:** Keeps count of your completed work sessions.
- **Visual Progress:** Dynamic progress bar and cycle indicators.
- **Responsive Design:** Works beautifully on desktop and mobile.
